# Phemex futures price downloader

Application for downloading current cryptocurrency prices from Phemex exchange defined in Excel for further use in other applications.

## How to build
```bash
./build.sh
```

## How to deploy
```
./deploy.sh
```

## How to prepare
```bash
./prepare.sh
```

## How to run
```bash
./run.sh
```

## Technologies
* Python 3
* Pandas
* CCXT
